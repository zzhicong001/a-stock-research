一、英伟达 AI PC

周末，英伟达PC的事情发酵较多。

英伟达、微软、ARM 官方账号同步预热 "PC 的新时代"，并附上一个看似指向台北的坐标。消息称下周台北 COMPUTEX 会上，英伟达将推出 N1 / N1X Windows-on-Arm 芯片，并和微软推出首批由英伟达芯片担任主处理器的 Windows PC，戴尔、联想也在跟进。

过去 Windows PC 的核心竞争主要围绕 x86 CPU 展开，GPU 更多是独立外挂，AI 能力则主要依赖云端或低功耗 NPU；而 N1X 的思路是把 CPU、GPU、NPU、统一内存和英伟达软件栈放在一起，让 PC 从传统生产力工具，逐步变成可以本地运行模型、执行 agent 工作流、承担低延迟 AI 推理的 "端侧 AI 主机"。

普通AI PC卖点是系统里有AI功能，而 N1X 核心卖点是 CUDA，本地有一套 NVIDIA AI 开发环境，200TOPS 可以在本地完整运行大模型。

目前看定位在小众高端市场，作为本地AI工具而不是传统PC换机，目标用户是 AI 开发者、本地模型玩家、对隐私和低延迟有要求的专业用户、需要 Windows 又要 CUDA 的用户等等。

郭明錤的供应链消息，未来两年 N1X 设备出货量约 1000 万台，是否上调主要取决于 Windows 是否能提供真正协调设备端 AI 计算的应用程序和工作流程。
